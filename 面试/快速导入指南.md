# Anki卡片快速导入指南（2分钟上手）

## 📥 一键导入步骤

### 1️⃣ 下载并安装Anki
- 访问：https://apps.ankiweb.net/
- 下载对应系统版本
- 安装并启动

### 2️⃣ 导入核心记忆卡片（选择题+简答题+代码题）

1. 快捷键 `Ctrl+Shift+I`（Mac: `Cmd+Shift+I`）
2. 选择文件：`C语言核心记忆.csv`
3. 导入设置：
   ```
   类型：基础（或带反转卡片）
   允许HTML：✅勾选
   字段分隔符：制表符
   ```
4. 点击"导入"

### 3️⃣ 导入填空记忆卡片

1. 快捷键 `Ctrl+Shift+I`
2. 选择文件：`C语言填空记忆.csv`
3. 导入设置：
   ```
   类型：挖空（Cloze）⚠️ 重要！
   允许HTML：✅勾选
   字段分隔符：制表符
   ```
4. 点击"导入"

---

## ✅ 导入成功标志

导入后，你应该看到：
- 牌组："C语言面试"
  - 子牌组："核心记忆"（30张卡片）
  - 子牌组："填空记忆"（24张卡片）

## 🚀 开始学习

## ⚙️ 推荐首次设置

1. 点击牌组右侧的⚙️图标

## ❓ 常见问题速查

### 问题1：导入后看不到卡片？
- 检查是否选对了"类型"（填空题必须选"挖空"）
- 重新导入，确保勾选"允许HTML"

### 问题2：代码显示乱码？
- 确保导入时"允许HTML"已勾选
- 字段分隔符选择"制表符"

### 问题3：每天卡片太多？
- 牌组设置 → "每天新卡片上限"改为10
- 可以暂停某些子牌组

## 🎯 学习计划建议（54张卡片）

### 第1周：核心概念（7天）
- 每天新卡片：8-10张（填空记忆优先）
- 每天复习：所有到期卡片
- 目标：完成填空记忆24张 + 核心记忆6张

### 第2周：深化理解（7天）
- 每天新卡片：5-6张（核心记忆）
- 每天复习：所有到期卡片
- 目标：完成全部54张卡片学习

### 第3-4周：巩固记忆
- 停止新卡片
- 每天复习到期卡片（约15-25张）
- 重点：保持90%以上正确率

### 面试前1周：冲刺
- 停止学习新卡片
- 快速过一遍所有卡片
- 重点复习标记为"困难"的卡片
- 结合面试真题.md深入理解

## 💪 学习技巧

1. **主动回忆**：看到题目先思考，实在不会再翻卡片
2. **理解为先**：不要死记硬背，理解核心原理
3. **填空优先**：填空题最适合快速记忆，先完成这部分
4. **代码动手**：代码题要在IDE里实际写一遍
5. **每天坚持**：10分钟也比不学强，保持连续学习
6. **标记难点**：答错的卡片添加"困难"标签，重点复习
7. **结合文档**：遇到不理解的，回到面试真题.md查看详细解释
