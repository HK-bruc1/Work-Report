## 三、数据格式

### （一）JL上位机在线调试协议

| Byte | offset | 备注 |
| ---- | ------ | ---- |
| | 0 | 数据包的长度（从offset=1起计算） |
| | 1 | 数据包类型 |
| | 2 | Seq，数据包序号，循环递增，取值范围（0~255） |
| | 3~N | Payload具体数据 |

**Tips:** 其中ENC指令，数据包类型为0X0B；seq可填1。数据格式为小端格式。

### （二）（新格式）Payload数据格式：MAGIC+CRC16+LEN+DATA

| MAGIC (u16) | CRC16 (u16) | CMD (1 Byte) | LEN (2 Byte) | DATA (长度不限) |
| ----------- | ----------- | ------------ | ------------ | --------------- |
| 标识符0X55CC | CRC16校验 | 命令号 | DATA长度 | 数据 |

**Tips：** CRC16（CRC-XModem）只需校验CMD+LEN+DATA的数据

### DATA命令信息

| DATA命令 | 功能说明 |
| -------- | -------- |
| 0x01 | 关闭算法，测试通话TALK MIC |
| 0x02 | 关闭算法，测试通话副(FF) MIC（需支持双麦通话降噪） |
| 0x03 | 开启通话算法 |
| 0x04 | 开启ANS/DNS降噪算法 |
| 0x05 | 关闭ANS/DNS降噪算法 |
| 0x06 | 测试TWS FF MIC 或者 头戴式/挂脖 左FF MIC |
| 0x07 | 测试TWS FB MIC 或者 头戴式/挂脖 左FB MIC |
| 0x08 | 测试头戴式/挂脖 右FF MIC |
| 0x09 | 测试头戴式/挂脖 右FB MIC |
| 0x0D | 关机指令 |
| 0x0E | 关闭算法，测试通话FB MIC |
| 0x0F | 双麦ENC MIC DIFF值下发 |
| 0x10 | 双麦ENC MIC CMP使能控制 |
| 0x11 | 双麦ENC MIC CMP补偿值清除记忆 |
| 0x12 | CVP算法单元独立控制 |
| 0x13 | DEC产测模式设置（bypass数据处理） |
| 0x14 | 设置MIC与目标金机MIC的差值 |
| 0x15 | 读取MIC与目标金机MIC的差值 |
| 0x16 | 关闭算法，测试通话骨传导VPU通路 |

### DATA反馈信息

| DATA命令 | 说明 |
| -------- | ---- |
| 0xA1 | 未知命令 |
| 0xA2 | CRC错误 |
| 0xA3 | VM写入错误 |
