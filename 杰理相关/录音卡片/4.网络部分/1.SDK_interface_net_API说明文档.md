## 一、Socket API

**头文件：** `lwip_2_2_0/lwip/app/sock_api/sock_api.h`

对 LwIP 原生 socket 的封装层，增加了超时回调、互斥锁保护等功能。

### 核心数据结构

| 结构体/枚举 | 说明 |
|---|---|
| `struct sock_hdl` | Socket 句柄，包含描述符、超时配置、互斥锁、回调函数 |
| `enum sock_api_msg_type` | 回调消息类型：连接成功/失败、收发超时、退出等 |

**`enum sock_api_msg_type` 枚举值：**

| 枚举值 | 说明 |
|---|---|
| `SOCK_SEND_TO` | 发送超时触发回调 |
| `SOCK_RECV_TO` | 接收超时触发回调 |
| `SOCK_EVENT_ALWAYS` | socket 操作时触发回调 |
| `SOCK_CONNECT_SUCC` | 连接成功触发回调 |
| `SOCK_CONNECT_FAIL` | 连接失败触发回调 |
| `SOCK_UNREG` | `sock_unreg` 时触发回调 |

### Socket 生命周期

| 函数 | 说明 |
|---|---|
| `sock_reg(domain, type, protocol, cb_func, priv)` | 创建 socket，分配句柄，注册回调。支持 `AF_INET`/`AF_INET6`；`SOCK_STREAM`(TCP)、`SOCK_DGRAM`(UDP)、`SOCK_RAW` |
| `sock_unreg(sock_hdl)` | 关闭 socket，释放句柄 |
| `sock_set_quit(sock_hdl)` | 设置退出标志 |
| `sock_get_quit(sock_hdl)` | 获取退出标志，返回 0 未退出，1 已退出 |
| `sock_clr_quit(sock_hdl)` | 清除退出标志 |
| `sock_get_socket(sock_hdl)` | 获取底层 socket 描述符 |

### TCP 服务端/客户端

| 函数 | 说明 |
|---|---|
| `sock_bind(sock_hdl, name, namelen)` | 绑定本地地址和端口 |
| `sock_listen(sock_hdl, backlog)` | 监听连接请求（仅 TCP 服务端） |
| `sock_accept(sock_hdl, addr, addrlen, cb_func, priv)` | 接受新连接，返回新的 `sock_hdl` |
| `sock_connect(sock_hdl, name, namelen)` | 发起 TCP 连接 |
| `sock_set_connect_to(sock_hdl, sec)` | 设置连接超时时间（秒） |

### 数据收发

| 函数 | 说明 |
|---|---|
| `sock_recv(sock_hdl, buf, len, flag)` | TCP 接收数据，返回字节数；0 表示断开，-1 表示错误 |
| `sock_send(sock_hdl, buf, len, flag)` | TCP 发送数据，返回字节数；0 表示断开，-1 表示错误 |
| `sock_recvfrom(sock_hdl, buf, len, flags, from, fromlen)` | UDP 接收数据，同时获取来源地址 |
| `sock_sendto(sock_hdl, data, size, flags, to, tolen)` | UDP 发送数据到指定目标地址 |

### 超时与状态控制

| 函数 | 说明 |
|---|---|
| `sock_set_recv_timeout(sock_hdl, millsec)` | 设置接收超时（ms） |
| `sock_set_send_timeout(sock_hdl, millsec)` | 设置发送超时（ms） |
| `sock_recv_timeout(sock_hdl)` | 查询是否发生接收超时 |
| `sock_send_timeout(sock_hdl)` | 查询是否发生发送超时 |
| `sock_would_block(sock_hdl)` | 查询 socket 是否处于阻塞状态，0:非阻塞，1:阻塞 |
| `sock_select(sock_hdl, readset, writeset, exceptset, tv)` | 多路复用等待（等同标准 `select`）；`tv=NULL` 永久等待，`tv=0` 立即返回 |
| `sock_select_rdset(sock_hdl)` | 接收前执行 select 检测 |

### 选项与控制

| 函数 | 说明 |
|---|---|
| `sock_fcntl(sock_hdl, cmd, val)` | 文件控制，支持 `F_GETFL`/`F_SETFL`，`O_NONBLOCK` 非阻塞标志 |
| `sock_getsockopt(sock_hdl, level, optname, optval, optlen)` | 获取 socket 选项 |
| `sock_setsockopt(sock_hdl, level, optname, optval, optlen)` | 设置 socket 选项 |
| `sock_set_reuseaddr(sock_hdl)` | 设置本地地址复用（`SO_REUSEADDR`） |
| `socket_set_keepalive(sock_hdl, keep_idle, keep_intv, keep_cnt)` | 设置 TCP keepalive 参数（idle 秒、间隔秒、重试次数） |
| `sock_get_error(sock_hdl)` | 获取最近一次操作的错误码 |
| `sock_getsockname(sock_hdl, name, namelen)` | 获取本地地址信息 |
| `sock_getpeername(sock_hdl, name, namelen)` | 获取对端地址信息 |

## 二、LwIP 网络栈管理

**头文件：** `lwip_2_2_0/lwip/port/lwip.h`

提供网络接口初始化、DHCP 管理、IP 地址查询等底层网络栈管理功能。

### 支持的网络接口类型

| 宏定义 | 值 | 说明 |
|---|---|---|
| `ETH_NETIF` | 0 | 有线以太网 |
| `WIFI_NETIF` | 1 | WiFi 无线 |
| `BT_NETIF` | 2 | 蓝牙网络（BT PAN） |
| `LTE_NETIF` | 3 | LTE/4G 蜂窝网络 |
| `EXT_WIFI_NETIF` | 4 | 扩展 WiFi |
| `WIFI_RAW_NETIF` | 5 | WiFi Raw 帧接口 |

### 网络接口生命周期

| 函数 | 说明 |
|---|---|
| `Init_LwIP(lwip_netif)` | 初始化指定网络接口的 LwIP 协议栈 |
| `lwip_netif_set_up(lwip_netif)` | 启用网络接口（Link Up） |
| `lwip_netif_set_down(lwip_netif)` | 禁用网络接口（Link Down） |
| `lwip_netif_remove(lwip_netif)` | 移除网络接口 |
| `lwip_set_default_netif(lwip_netif)` | 设置系统默认路由网络接口 |

### DHCP 管理

| 函数 | 说明 |
|---|---|
| `lwip_renew(lwip_netif, dhcp)` | 更新网络配置，可选是否重新发起 DHCP |
| `lwip_dhcp_bound()` | 查询 DHCP 是否已绑定成功，返回 0/1 |
| `lwip_dhcp_release_and_stop(lwip_netif)` | 释放 DHCP 租约并停止 DHCP 客户端 |
| `lwip_set_dhcp_timeout(sec)` | 设置 DHCP 超时时间（秒） |

### 地址查询

| 函数 | 说明 |
|---|---|
| `Get_IPAddress(lwip_netif, ipaddr)` | 获取指定接口的 IP 地址字符串 |
| `get_gateway(lwip_netif, ipaddr)` | 获取网关地址字符串 |
| `gethostname(name, namelen)` | 获取本机主机名 |
| `lwip_get_netif_info(is_wireless, info)` | 获取 `netif_info`（IP/网关/子网掩码） |
| `get_netif_macaddr_and_ipaddr(ipaddr, mac_addr, count)` | 批量获取所有接口的 IP 和 MAC 地址 |
| `lwip_get_dest_hwaddr(lwip_netif, ipaddr, dhwaddr)` | 通过 ARP 获取目标 IP 的 MAC 地址 |

### LAN 配置

| 函数 | 说明 |
|---|---|
| `net_get_lan_info(lwip_netif)` | 获取 LAN 配置（IP/掩码/网关/DHCP 服务器/起始地址） |
| `net_set_lan_info(lan_setting)` | 设置 LAN 配置（使用默认接口） |
| `net_set_lan_info_ext(lan_setting, lwip_netif)` | 设置 LAN 配置（指定接口） |
| `set_wireless_netif_macaddr(mac_addr)` | 设置无线接口 MAC 地址 |

### 其他

| 函数 | 说明 |
|---|---|
| `lwip_event_cb(lwip_ctx, event)` | LwIP 事件回调处理（DHCP 成功/超时等） |
| `LwIP_Pkt_Handle()` | 手动触发 LwIP 数据包处理 |
| `ip4_route2(src, dest)` | 根据源/目的 IP 进行策略路由查找 |
| `memp_get_pbuf_pool_free_cnt()` | 获取 pbuf 内存池剩余数量，用于监控内存压力 |

### LWIP 事件类型（`enum LWIP_EVENT`）

| 事件 | 说明 |
|---|---|
| `LWIP_WIRELESS_DHCP_BOUND_SUCC` | 无线网卡 DHCP 绑定成功 |
| `LWIP_WIRELESS_DHCP_BOUND_TIMEOUT` | 无线网卡 DHCP 绑定超时 |
| `LWIP_WIRE_DHCP_BOUND_SUCC` | 有线网卡 DHCP 绑定成功 |
| `LWIP_WIRE_DHCP_BOUND_TIMEOUT` | 有线网卡 DHCP 绑定超时 |
| `LWIP_LTE_DHCP_BOUND_SUCC` | LTE DHCP 绑定成功 |
| `LWIP_LTE_DHCP_BOUND_TIMEOUT` | LTE DHCP 绑定超时 |
| `LWIP_BT_DHCP_BOUND_SUCC` | 蓝牙网络 DHCP 绑定成功 |
| `LWIP_BT_DHCP_BOUND_TIMEOUT` | 蓝牙网络 DHCP 绑定超时 |
| `LWIP_BT_DHCP_BOUND_RELEASE` | 蓝牙网络 DHCP 租约释放 |

## 三、WebSocket API

**头文件：** `web_socket/websocket_api.h`

支持 WebSocket 客户端和服务端，含 TLS（mbedTLS）加密通信（WSS）。

### 核心数据结构

| 结构体 | 说明 |
|---|---|
| `WEBSOCKET_INFO` | WebSocket 连接实例，包含 socket 句柄、IP/端口、收发缓冲区、回调函数指针、TLS 上下文等 |
| `WEBSOCKETS_MBTLS_INFO` | TLS 安全上下文，封装 mbedTLS 的证书、SSL、熵源、随机数等 |
| `struct websocket_req_head` | HTTP 升级握手请求头（方法、路径、Host、版本） |

### 数据帧类型（`WS_CMD_Type`）

| 枚举值 | 值 | 说明 |
|---|---|---|
| `WCT_SEQ` | 0x00 | 序列数据包 |
| `WCT_TXTDATA` | 0x01 | 文本数据帧（UTF-8） |
| `WCT_BINDATA` | 0x02 | 二进制数据帧 |
| `WCT_DISCONN` | 0x08 | 断开连接帧 |
| `WCT_PING` | 0x09 | Ping 心跳帧 |
| `WCT_PONG` | 0x0A | Pong 心跳帧 |
| `WCT_FIN` | 0x80 | FIN 结束标志 |
| `WCT_CLOSE_OK` | 0xAA | 关闭确认 |

### 连接状态（`WS_STATUS`）

| 枚举值 | 说明 |
|---|---|
| `NOT_ESTABLISHED` | 未建立连接 |
| `ESTABLISHED` | 连接已建立 |
| `INVALID_ESTABLISHED` | 连接无效 |

### WebSocket 客户端

| 函数 | 说明 |
|---|---|
| `websockets_client_socket_init(websocket_info)` | 初始化客户端，建立 TCP/TLS 连接 |
| `webcockets_client_socket_handshack(websocket_info)` | 执行 WebSocket HTTP 升级握手 |
| `websockets_client_socket_send(websocket_info, buf, len, type)` | 客户端发送数据帧 |
| `websockets_client_socket_recv(websocket_info)` | 客户端接收数据帧 |
| `websockets_client_socket_exit(websocket_info)` | 断开客户端连接并释放资源 |
| `websockets_client_notify_disconnet_to_server(websockets_info)` | 向服务端发送 WebSocket Disconnect 通知帧 |
| `websockets_client_socket_heart_thread(param)` | 客户端心跳保活线程（Ping/Pong） |
| `websockets_client_socket_recv_thread(param)` | 客户端数据接收线程 |

### WebSocket 服务端

| 函数 | 说明 |
|---|---|
| `websockets_serv_socket_init(websockets_info)` | 初始化服务端，创建监听 socket |
| `websockets_serv_socket_hanshack(websockets_info)` | 处理客户端升级握手请求 |
| `websockets_serv_socket_send(websockets_info, buf, len, type)` | 服务端发送数据帧 |
| `websockets_serv_socket_recv(websockets_info)` | 服务端接收数据帧 |
| `websockets_serv_socket_exit(websockets_info)` | 关闭服务端连接 |
| `websockets_serv_socket_heart_thread(param)` | 服务端心跳保活线程 |

### 通用工具函数

| 函数 | 说明 |
|---|---|
| `websockets_socket_send(websockets_info, buf, len, type)` | 通用数据发送，`type` 可自定义帧类型 |
| `websockets_ping_heart_beat(websockets_info, buf, index)` | 发送 Ping 心跳包 |
| `websockets_pong_heart_beat(websockets_info, buf, len, index)` | 发送 Pong 心跳响应包 |
| `websocket_msg_fifo(websockets_info, msg_value)` | 消息入队（环形 FIFO） |
| `websocket_msg_get(websockets_info)` | 消息出队 |
| `websocket_msg_clear(websockets_info)` | 清空消息队列 |
| `websockets_send_data_set_seq_packet(websockets_info, seq_en)` | 设置是否使用序列包模式发送 |
| `websockets_struct_check(sizeof_struct)` | 检查结构体大小一致性（防止配置错误） |
| `websockets_sleep(ms)` | WebSocket 内部延时函数（ms） |

## 四、WiFi 连接管理

**头文件：** `wifi/wifi_connect.h`

### WiFi 工作模式（`enum WIFI_MODE`）

| 枚举值 | 说明 |
|---|---|
| `STA_MODE` | Station 客户端模式，连接到 AP |
| `AP_MODE` | 热点模式，充当接入点 |
| `SMP_CFG_MODE` | 配网/Monitor 模式（Airkiss 等） |
| `MP_TEST_MODE` | 产线测试模式 |
| `P2P_MODE` | WiFi P2P 直连模式 |
| `NONE_MODE` | 无模式 |

### 加密类型

| 宏 | 说明 |
|---|---|
| `ENC_NONE` | 无加密（开放网络） |
| `ENC_WEP` | WEP 加密 |
| `ENC_WPA` | WPA 加密 |
| `ENC_WPA2` | WPA2 加密 |
| `ENC_WPA3` | WPA3 加密 |

### WiFi 事件（`enum WIFI_EVENT`）

| 事件 | 说明 |
|---|---|
| `WIFI_EVENT_MODULE_INIT/START/STOP` | 模块初始化/启动/停止 |
| `WIFI_EVENT_STA_START/STOP` | STA 模式启动/停止 |
| `WIFI_EVENT_STA_CONNECT_SUCC` | STA 连接成功 |
| `WIFI_EVENT_STA_CONNECT_TIMEOUT_NOT_FOUND_SSID` | 未找到目标 SSID |
| `WIFI_EVENT_STA_CONNECT_ASSOCIAT_FAIL/TIMEOUT` | 关联失败/超时 |
| `WIFI_EVENT_STA_DISCONNECT` | STA 断开连接 |
| `WIFI_EVENT_STA_NETWORK_STACK_DHCP_SUCC/TIMEOUT` | DHCP 获取 IP 成功/超时 |
| `WIFI_EVENT_STA_SCAN_COMPLETED` | 扫描完成 |
| `WIFI_EVENT_AP_START/STOP` | AP 模式启动/停止 |
| `WIFI_EVENT_AP_ON_ASSOC` | 有 STA 接入 |
| `WIFI_EVENT_AP_ON_DISCONNECTED` | 有 STA 断开 |
| `WIFI_EVENT_SMP_CFG_START/STOP/TIMEOUT/COMPLETED` | 配网开始/停止/超时/成功 |
| `WIFI_EVENT_P2P_START/STOP` | P2P 启动/停止 |
| `WIFI_EVENT_PM_SUSPEND/RESUME` | 省电模式挂起/恢复 |

### 基础控制

| 函数 | 说明 |
|---|---|
| `wifi_on()` | 启动 WiFi 模块 |
| `wifi_off()` | 关闭 WiFi 模块 |
| `wifi_is_on()` | 查询 WiFi 是否已启动 |
| `wifi_set_event_callback(cb)` | 注册 WiFi 事件回调函数 |
| `wifi_get_mac(mac)` | 获取 WiFi MAC 地址（6字节数组） |
| `wifi_set_mac(mac_addr)` | 设置 WiFi MAC 地址 |
| `wifi_set_pwr(pwr_sel)` | 设置发射功率等级，范围 0（最低）~6（默认最高） |
| `wifi_get_mode_cur_info(info)` | 获取当前 WiFi 模式及 SSID/密码信息 |
| `wifi_get_mode_stored_info(info)` | 获取最后记忆的 WiFi 模式配置 |
| `wifi_set_default_mode(parm, force, store)` | 设置 WiFi 默认启动模式和配置 |
| `wifi_store_mode_info(mode, ssid, pwd)` | 保存 WiFi 模式配置，覆盖默认记忆 |

### STA 模式（客户端）

| 函数 | 说明 |
|---|---|
| `wifi_enter_sta_mode(sta_ssid, sta_pwd)` | 进入 STA 模式并连接指定网络 |
| `wifi_set_sta_connect_timeout(sec)` | 设置连接超时时间（秒，默认 60s） |
| `wifi_set_connect_sta_block(block)` | 设置连接为阻塞（1）/非阻塞（0）模式 |
| `wifi_set_sta_connect_best_ssid(enable)` | 开启自动连接历史中信号最强的网络 |
| `wifi_set_store_ssid_cnt(cnt)` | 设置最多保存的 SSID 数目（默认 5） |
| `wifi_get_sta_connect_state()` | 获取 STA 连接状态（枚举 `wifi_sta_connect_state`） |
| `wifi_get_rssi()` | 获取当前连接的信号强度（RSSI，有符号值） |
| `wifi_get_cqi()` | 获取信道质量指数，0-100（>50 较好，20-50 一般，<20 较差） |
| `wifi_get_stored_sta_info(info[])` | 获取历史已保存的 SSID 列表 |
| `wifi_del_stored_sta_info(ssid)` | 删除指定的已保存 SSID |

### AP 模式（热点）

| 函数 | 说明 |
|---|---|
| `wifi_enter_ap_mode(ap_ssid, ap_pwd)` | 进入 AP 热点模式 |
| `wifi_get_sta_entry_rssi(wcid, rssi, evm, mac)` | 获取已接入 STA 的信号强度和 MAC 地址 |
| `wifi_disconnect_station(mac, reason)` | 断开指定 MAC 地址的 STA 连接（reason=8 标准离开） |

### 扫描

| 函数 | 说明 |
|---|---|
| `wifi_scan_req()` | 发起一次 SSID 扫描（STA 或 AP 模式均可） |
| `wifi_get_scan_result(ssid_cnt)` | 获取扫描结果列表，返回 `wifi_scan_ssid_info` 数组指针 |
| `wifi_clear_scan_result()` | 清空上次扫描结果，释放内存 |

### 配网模式（SmpCfg / Airkiss）

| 函数 | 说明 |
|---|---|
| `wifi_enter_smp_cfg_mode()` | 进入配网/Monitor 模式 |
| `wifi_set_smp_cfg_timeout(sec)` | 设置配网超时时间（秒） |
| `wifi_set_smp_cfg_scan_all_channel(onoff)` | 是否扫描全部信道（1:扫全部，0:只扫当前） |
| `wifi_set_smp_cfg_airkiss_recv_ssid(onoff)` | 是否接收完整 SSID（开启加速配网但可能失败） |
| `wifi_set_smp_cfg_just_monitor_mode(onoff)` | 进入配网后是否仅保留 Monitor 模式 |
| `wifi_set_monitor_mode_scan_channel_time(time_ms)` | Monitor 模式下每个信道的扫描间隔（ms） |
| `wifi_set_airkiss_key(key[16])` | 设置 Airkiss 加密 Key（16字节） |
| `wifi_get_cfg_net_result(smp_cfg)` | 配网成功后获取 SSID、密码及校验信息 |

### WSC（WPS）模式

| 函数 | 说明 |
|---|---|
| `wifi_wsc_start(ssid, pincode, mode)` | 启动 WPS 配置，支持 PIN/PBC 模式（`enum WSC_CFG_MODE`） |

### 底层/高级接口

| 函数 | 说明 |
|---|---|
| `wifi_get_channel()` | 获取当前工作信道 |
| `wifi_set_channel(ch)` | 设置当前工作信道 |

## 总结

| 模块 | 头文件路径 | 核心用途 |
|---|---|---|
| **Socket API** | `lwip_2_2_0/lwip/app/sock_api/sock_api.h` | 带超时/回调的 TCP/UDP Socket 封装 |

| **mbedTLS** | `mbedtls_3_4_0/mbedtls/*.h` | TLS 安全通信、对称/非对称加密、证书管理 |
| **PSA Crypto** | `mbedtls_3_4_0/psa/crypto.h` | PSA 统一加密 API（mbedTLS 3.x 推荐方式） |
